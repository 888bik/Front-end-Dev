# 文本对象
在 vim 中，文本是结构化的存在，即 文本对象（text—object）；而 vim 对于文本对象，专门有一部分语法，用于快速对文本对象进行编辑操作。
# 语法结构
文本对象的操作语法：operater + 范围（内 / 外）+ 文本对象
内部：i ，意指 inner
外部：a ，英文单词 a，一个的意思
